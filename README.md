## UserManager — Декларативная обработка коллекций (Stream API)

### Функциональность
`getAdults()` - Возвращает пользователей старше 18 лет
`getITUserNames()` - Извлекает имена сотрудников отдела IT
`getAverageAge()` - Вычисляет средний возраст
`getYoungestUser()` - Находит самого молодого пользователя
`groupByDepartment()` - Группирует пользователей по отделам
